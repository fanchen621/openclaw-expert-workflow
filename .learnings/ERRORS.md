# Error Log

> 记录命令失败、异常和工具错误。
> 定期审查以识别反复出现的问题。

## 格式

```markdown
### [日期] - [工具/命令] - [简述]
- **命令**: 执行了什么
- **错误输出**: 关键错误信息
- **根因**: 为什么会失败
- **修复**: 怎么解决的
- **是否复发**: yes/no
```
