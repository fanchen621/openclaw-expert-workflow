# Learnings Log

> 记录纠正、知识差距和最佳实践。
> 每次出错或学到新东西时追加条目。

## 格式

```markdown
### [日期] - [类别] - [简述]
- **上下文**: 发生了什么
- **根因**: 为什么会发生
- **修复**: 怎么解决的
- **预防**: 未来如何避免
- **See Also**: 相关条目链接
```

## 类别
- `correction` — 用户纠正了我
- `knowledge_gap` — 知识过时或缺失
- `best_practice` — 发现了更好的方法
- `pattern` — 发现了可复用的模式
